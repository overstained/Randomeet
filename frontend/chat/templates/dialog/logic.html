IQ(Intelligence quotient) is a score obtained through stadardized tests designed to quantify human intelligence.
Raven's Progressive Matrices is a non-verbal test used as an estimate for fluid intelligence, which is the capacity to solve novel problems by analyzing, indentifying patterns, relationships and extrapolation of these using logic.
